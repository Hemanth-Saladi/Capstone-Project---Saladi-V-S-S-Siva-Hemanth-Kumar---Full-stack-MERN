# 🎵 Music Library – Full Stack MERN

## Features

### User
- Register / Login / Logout
- View songs
- Search songs
- Create playlists
- Add songs to playlist
- Like songs
- Play / Pause / Queue / Next

### Admin
- Admin login
- Add songs
- Add artists
- Delete songs
- View analytics
- Restrict songs

## Tech Stack

Frontend: React  
Backend: Node.js + Express  
Database: MongoDB  
Auth: JWT  
Storage: Local Storage  

## How to run

Backend:
npm install
npm run dev

Frontend:
npm install
npm start
