Sarva-Gyani – Running Guide


1. "Create a new React application" in your local system.
npx create-react-app sarva-gyani


2. Navigate into the project folder.
cd sarva-gyani

3. Replace the default `src` folder generated by Create React App with the `src` folder provided in this submission.

4. Install the required dependency.
npm install bootstrap


5. Start the React development server.
npm start


6. The application will run at:

http://localhost:3000


Notes

Ensure the provided src folder structure remains unchanged after replacement.

The project demonstrates advanced React concepts including 

Lazy Loading, Pure Components, Error Boundaries, and React Portals.

The interface represents a simplified "LMS dashboard (Sarva-Gyani)" designed for MERN stack learning.
